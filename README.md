**Perfect Automation** is a lightweight tech-focused modpack built on top of [Fabulously Optimized](https://modrinth.com/modpack/fabulously-optimized) for excellent performance and visuals, with added automation and machinery from **[Oritech](https://modrinth.com/mod/oritech)** and a few supporting mods.

Designed for small servers or friends who want smooth gameplay while experimenting with Oritech's animated machines, advanced processing chains, electric tools/armor, drones, lasers, and more — without heavy kitchen-sink bloat.

Minecraft version: 1.21.1
Loader: Fabric 
Environment: Client + Server compatible

This is a modified fork of Fabulously Optimized — huge thanks to the FO team!
